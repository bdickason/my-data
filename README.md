my-data
=======

### Create your own personal API

* Input data by hand (key/value pair) 
* Send your data via API (key/value pair)
* Permission your data (let others access it)
* Graph your data


### Example:

```
public: {
  email: "dickason@gmail.com",
  name: "Brad Dickason",
  addresses: [
    work: {
      street: "blah"
    }

  ]
},

private: {
  phone: "212-212-2121",
  middle-name: "Thomas",
  addresses: [
    home: {
      street: "blah"
    }
  ]
}
```
